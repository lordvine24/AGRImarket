Agri Market

Agri Market is a Flutter-based mobile application designed to connect farmers and buyers in an online marketplace, providing a seamless platform for buying and selling agricultural products. The app aims to bridge the gap between farmers and consumers, promoting fair trade, transparency, and convenience.

Project Aim

The main goal of Agri Market is to create a digital space where:

Farmers can showcase their fresh produce and reach a wider audience.

Buyers can discover, compare, and purchase products directly from local farmers.

Transactions and communication happen efficiently in a user-friendly online environment.

By connecting producers and consumers, Agri Market empowers farmers economically and makes fresh agricultural products more accessible to buyers.

Features

User Registration & Authentication: Secure login for both farmers and buyers.

Product Listings: Farmers can add and manage product listings with descriptions, images, and prices.

Search & Filter: Buyers can search for products and filter by type, location, or price.

Interactive Forms: Input fields, dropdowns, and switches for easy user interaction.

Custom UI Components: Reusable buttons, cards, and input fields for a consistent user experience.

Animations & Transitions: Smooth page navigation and animated success messages for better UX.

Form Validation: Ensures data accuracy and prevents invalid input.

Technologies Used

Getting Started

Clone the repository

Install dependencies

flutter pub get


Run the app

flutter run
